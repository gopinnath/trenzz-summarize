# Introduction

Trenzz Summarize is sample application that is developed to showcase the capabilities of the Quarkus. The application uses Quarkus, Quarkus Resteasy, Quarkus Rest Client, Quarkus Scheduler, Quarkus Hibernate Orm, and Quarkus JDBC H2.

This application also has docker

## About the Usecase

The usecase here is application that captures the Top 10 Trending keywords from Twitter for give place and displays that as a graphs on UI. HTML, Java Script and D3 Framework are used for UI rendering.
